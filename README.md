# ALEX.DEV Portfolio - Neural Flux v2.0

A cutting-edge AI Engineer portfolio built with React, Three.js, and Framer Motion. This project showcases a "Neural Flux" design aesthetic with immersive 3D backgrounds and interactive elements.

![Portfolio Preview](public/vite.svg)

## ‚ú® Features

- **Neural Background**: Interactive 3D particle system reacting to mouse and scroll (Three.js/R3F).
- **Magnetic Experience**: Custom arrow cursor with physics-based trail and magnetic hover effects.
- **Orbital Navigation**: Unique circular navigation system that tracks scroll progress.
- **Smooth Scrolling**: Implemented using Lenis for a premium feel.
- **Dynamic Content**:
  - **Constellation Projects**: Grid layout with connection lines.
  - **Skills Orbit**: 3D orbiting skill visualization.
  - **Experience Timeline**: Vertical timeline with glassmorphism cards.
  - **Contact Portal**: Interactive contact section.
